# Celsus to Fahrenheit conversion

Convert degrees Celsius to degrees Fahrenheit

## Usage

``` r
C_to_F(C_temp)
```

## Arguments

- C_temp:

  The temperature in degrees Celsius

## Value

The temperature in degrees Fahrenheit

## Examples

``` r
temp1 <- C_to_F(22);
temp2 <- C_to_F( c(-2, 12, 23));
```
