# Date formats

A list of possible date formats.

## Usage

``` r
date_formats
```

## Format

A list of vectors

- mdy:

  Vector of month-day-year formats

- yyyymd:

  Vector of year-month-day formats

- bmo:

  Vector with letter or word months

- dmy:

  Vector of day-month-year formats

- unlikely:

  Vector of unlikely date formats
